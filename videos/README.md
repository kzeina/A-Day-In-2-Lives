Place your video files here with these exact filenames:

| Filename              | Description               |
|-----------------------|---------------------------|
| hero.mp4              | Landing page background video (any ratio, autoplay) |
| avani-morning.mp4     | Avani's morning segment   |
| avani-afternoon.mp4   | Avani's afternoon segment |
| avani-evening.mp4     | Avani's evening segment   |
| asemai-morning.mp4    | Asemai's morning segment  |
| asemai-afternoon.mp4  | Asemai's afternoon segment|
| asemai-evening.mp4    | Asemai's evening segment  |

All story videos should be 16:9 horizontal ratio (e.g. 1920×1080, 1280×720).
The hero video can be any ratio — it will be cropped to fill the screen.
